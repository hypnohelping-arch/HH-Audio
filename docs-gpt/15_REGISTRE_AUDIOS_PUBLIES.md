# Registre des audios publiés
Rôle : consigner les audios livrés, leurs métadonnées et leur statut de conformité.

## Règles clés
- Enregistrer pour chaque audio : titre, mode, version, date, statut LOCKED.
- Mentionner la source de script et les validations QA associées.
- Indiquer les licences musicales ou sonores utilisées.
- Archiver les liens de distribution et les retours utilisateurs synthétiques.
- Mettre à jour dès qu'un audio est retiré ou corrigé.
