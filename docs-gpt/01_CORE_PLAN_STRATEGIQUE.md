# Core — Plan stratégique (miroir canon /docs/audio/PLAN_REFERENCE_USINE_AUDIO_PREMIUM+++.md)

## Rôle
Tracer l’objectif stratégique de l’usine audio Premium+++ : automatisation maximale, qualité hypnotique irréprochable, sécurité émotionnelle absolue et scalabilité sans dégrader la voix, le langage ou l’expérience.

## Axes canoniques
- Pipeline unique pour tous les audios, avec deux modes : STANDARD Premium+++ et SIGNATURE (mêmes étapes, validation renforcée).
- Textes hypnotiques souverains : scripts clean/studio hébergés dans chaque projet GitHub, sans indications musicales/techniques.
- Journal perceptif humain obligatoire par bloc (mémoire qualitative, jamais automatisé).
- Aucun contournement d’étape critique ; toute évolution passe par décision explicite et traçable.

## Gouvernance des autorités (canon)
- ChatGPT : architecte, spécification des règles/workflows et validation conceptuelle.
- Codex : exécution automatisée strictement déterministe, sans initiative créative.
- GitHub : canon texte (scripts, manifests, registres, QA, décisions).
- pCloud : canon binaire (voix WAV, bounces, masters), périmètre restreint et idempotent.
