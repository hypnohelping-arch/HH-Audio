# Format Script Fractionné V1 (miroir canon /docs/audio/PLAN_REFERENCE_USINE_AUDIO_PREMIUM+++.md)

## Rôle
Définir le format unique et obligatoire des scripts d’hypnose audio pour garantir sécurité émotionnelle, confort vocal et industrialisation sans dégradation.

## Principes canoniques
- Deux versions obligatoires par projet : `script_clean.md` (texte hypnotique pur) et `script_studio.md` (intentions vocales, silences, respiration).
- Aucune indication musicale, sonore ou technique n’est autorisée dans le texte hypnotique.
- Découpage long et fonctionnel en blocs hypnotiques ; chaque bloc correspond à une fonction claire et devient l’unité de travail audio/musical.
- La segmentation est figée avant tout sound design.

## Gouvernance
- Toute modification de script crée une nouvelle version audio liée explicitement (script_version ↔ audio_version).
- Les scripts vivent dans le projet audio GitHub, jamais dans pCloud.
