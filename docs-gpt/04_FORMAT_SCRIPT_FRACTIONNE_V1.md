# Format script fractionné v1
Rôle : standardiser la structure en segments d'un script d'hypnose fractionné.

## Règles clés
- Découper en unités numérotées : ouverture, induction, approfondissement, retour.
- Chaque segment commence par un identifiant unique et un objectif clair.
- Les balises de timing sont exprimées en secondes, non en mesures musicales.
- Les transitions doivent inclure un signal explicite de consentement continu.
- Toute insertion sonore est marquée PENDING tant que non validée par QA.
