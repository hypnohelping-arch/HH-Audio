# Modes de séance — Core
Rôle : lister et cadrer les formats de séance disponibles dans le catalogue premium.

## Règles clés
- Décrire chaque mode : objectif, durée cible, segments obligatoires.
- Associer un niveau d'engagement utilisateur et une intensité sonore cible.
- Prévoir un protocole de sortie sécurisé pour chaque mode.
- Identifier les modes incompatibles avec certaines fréquences binaurales.
- Archiver les retours QA pour ajuster les modes au fil des itérations.
