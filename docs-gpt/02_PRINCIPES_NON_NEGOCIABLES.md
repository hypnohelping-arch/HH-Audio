# Principes non négociables
Rôle : verrouiller les garde-fous éthiques et qualitatifs de toute séance.

## Règles clés
- Consentement explicite requis pour chaque progression hypnotique.
- Transparence sur les objectifs et la nature non médicale des contenus.
- Aucune promesse de résultat ; uniquement des accompagnements possibles.
- Confidentialité : aucune donnée personnelle dans les scripts.
- Droit de pause : chaque section doit prévoir une sortie sécurisée.
