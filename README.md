# HH-Audio

## Objet
Usine de production d'audios d'hypnose premium, structurée pour sécuriser la conception, l'enregistrement et la validation.

## Contenu du dépôt
- Gouvernance, chartes, registres et modèles de travail.
- Spécifications de production et bibliothèques de blocs non audio.
- Scripts et templates pour orchestrer les sessions.

## Ce que le dépôt ne contient pas
- Aucun audio final ou fichier volumineux enregistré.

## Workflow des statuts
DRAFT → RECORDED → VALIDATED → LOCKED

## Hiérarchie non négociable
1. Sécurité émotionnelle
2. Voix
3. Langage hypnotique
4. Musique
5. Effets

## Verrous anti-prolifération
- docs-gpt/ = corpus fermé (aucun nouveau fichier autorisé)
- 1 audio = 1 dossier session
- Backlog unique
- Templates immuables
- Aucun fichier audio lourd dans GitHub
